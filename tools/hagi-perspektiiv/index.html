Metoodika

Metoodika aitab Sul nõude eelanalüüsi teha viie elemendi kaupa: kronoloogia, rikkumine, põhjuslik seos, arvutatav kahju ja tõendid.

Oluline märkus

Tööriist aitab struktureerida mõtteid ja hinnata riske. See ei anna siduvat õiguslikku hinnangut ega garanteeri menetlustulemust. Kohtusse pöördumine sisaldab alati menetluskulude riski.
